## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS + shadcn/ui
- React Router v6
- Framer Motion (page transitions)
- Supabase (backend)

## Development

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:8080`.

## Build

```bash
npm run build
```

The production build outputs to the `dist/` directory.

## Deployment

Static site deployed to Cloudflare Workers. Build command: `npm run build`. Publish directory: `dist`.
